# learning-go

A project-driven learning lab for Go. Instead of reading through books end to end, the plan here is to combine a knowledge map with a sequence of progressively harder small projects.

## Navigation

- [roadmap.md](roadmap.md) — the knowledge map. What to learn, organized by stage.
- [projects/README.md](projects/README.md) — the action plan. Twelve projects from CLI tool to performance lab, with recommended order and iteration method.

Each project has its own folder under `projects/` with a self-contained README.
